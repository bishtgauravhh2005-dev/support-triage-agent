
# Multi-Domain Support Triage Agent

## Overview
A terminal-based NLP system that classifies and routes support tickets across multiple domains.

## Features
- Request type classification
- Product area detection
- Risk assessment
- Semantic retrieval using embeddings
- Safe escalation logic

## Tech Stack
- Python
- Sentence Transformers
- Scikit-learn
- Pandas

## How to Run
```bash
pip install -r requirements.txt
python main.py
